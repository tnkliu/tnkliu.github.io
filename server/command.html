服务器

Linux命令

查看某端口的连接数

netstat -nat | grep 6379 | wc -l

查看 TCP 连接情况

netstat -n | awk '/^tcp/ {++S[$NF]} END {for(a in S) print a, S[a]}'

按照内容搜索文件(其中 ./ 是服务器的目录)

grep "搜索关键字" -r ./

按照文件名搜索文件

find ./ -name filename

搜索并删掉

ps -ef | grep 'nginx' | awk '{print $2}' | xargs kill -9

查看系统版本

uname -r
cat /etc/issue
cat /proc/version

查看具体命令放在什么位置

which nginx

修改内核可以同时打开的文件描述符的最大值

ulimit -SHn 65535

查看某一端口占用情况

lsof -i :11211

查看内存条数

dmidecode -t memory | grep Size

修改服务器主机名

hostnamectl set-hostname myserver

设置ssh密钥

ssh-keygen -t rsa -C "your name"

tar 加密

tar -zcvf - xxx | openssl des3 -salt -k 123321 | dd of=xxx.des3

tar 解密

dd if=xxx.des3 | openssl des3 -d -k 123321 | tar zxf -

对文件内容排序(根据每行前面6个字符)

sort -k1.1,1.6 no.txt > ok.txt

监测网络中DNS服务器是否能正确实现域名解析

nslookup xxx.com

递归计数某文件夹下面有多少文件

find ./ -type f | wc -l

按照文件大小从小到大排序(Sr代表从小到大，S代表从大到小)

ls -lhaSr

根据文件夹大小按照从大到小排序

du | sort -nr | more
