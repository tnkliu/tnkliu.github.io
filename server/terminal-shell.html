快捷的zsh和fish shell

把终端升级了一下，经常操作的平台，唯快不破，索性分析了一下zsh和fish两个shell工具，两个都介绍一下。

终端，你需要通过它操作系统，记得从Windows切换到Linux的时候，对命令行又是神秘又是敬畏，怕不会操作，怕有误操作，但是熟悉后就能自主的控制文件，运行命令，可以自定义很多功能，慢慢熟悉后会非常喜欢，会越来越少的误操作，越来越知道什么是命令之美，需要多操作，多使用。

  • zsh shell
  1. yum命令可以直接安装
    yum install zsh
  2. 安装好以后执行
    chsh -s /bin/zsh
  3. 安装 oh-my-zsh 设置zsh主题
    sh -c “$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)”
  4. 安装 autojump 以便实现 j xxx 快速跳转到目录
    yum install autojump-zsh
  • fish shell
  1. 先到官网下载安装包 http://fishshell.com
  2. 安装好以后执行
    chsh -s /usr/local/bin/fish
  3. 重启终端
  4. 运行 fish_config 会打开一个网页，可以在浏览器中设置终端配色等
  5. 安装 oh-my-fish 可以通过 https://github.com/oh-my-fish/oh-my-fish 查看具体详情
    curl -L https://github.com/oh-my-fish/oh-my-fish/raw/master/bin/install > install
    fish install
  6. 切换主题，输入 omf theme 可以查看所有的主题，然后直接后面接对应的主题回车即可，比如 omf theme lambda，如果提示需要安装，直接omf install对应的主题即可
  7. 如果有自定义环境变量，需要按照如下方式定义
    set -x PATH $PATH /my/bin
  8. 好好享受fish shell带来的快捷把。
